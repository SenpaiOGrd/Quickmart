# Quickmart
 A MERN-stack grocery marketplace where users can easily buy and sell fresh groceries online. Featuring secure authentication, real-time inventory updates, intuitive search, and a smooth checkout process, it connects local sellers with customers for a convenient, fast, and reliable shopping experience.
